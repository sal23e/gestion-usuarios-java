# Gestión de Usuarios – Aplicación Java

Proyecto académico DAM para la gestión de usuarios con operaciones CRUD y conexión a base de datos MySQL.

## Tecnologías
- Java
- MySQL
- JDBC
- IntelliJ IDEA
- Git/GitHub

## Funciones
- Añadir, listar, editar y eliminar usuarios
- Conexión con base de datos MySQL
- Menú interactivo simple (consola)
